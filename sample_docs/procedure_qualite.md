# Procédure de Qualité - Contrôle des Pièces Usinées

Révision : 2.4
Date d'application : 01 Mars 2024

## Objectif
S'assurer que toutes les pièces usinées respectent les tolérances géométriques spécifiées dans le cahier des charges client (norme ISO 2768-m).

## Étapes de Contrôle
1. Échantillonnage : Prélever 5 pièces par lot de 100.
2. Mesure : Utiliser le micromètre digital étalonné (réf: M-092) pour les diamètres extérieurs.
3. Conformité : La tolérance acceptée est de +/- 0.05 mm. Toute pièce hors tolérance entraîne le rejet complet du lot.

## Traitement des Non-Conformités
Créer un ticket "NC" (Non-Conformité) dans le système. Isoler physiquement le lot dans la zone rouge.
